# TensorTales

Personal blog on machine learning, statistics, and deep learning.

**Live site:** [markkurzeja.github.io](https://markkurzeja.github.io)

## Build

Requires Python 3.13+ and [uv](https://docs.astral.sh/uv/).

```bash
uv run python build.py
```

This converts Markdown posts in `src/` to static HTML using a Tufte-inspired theme with KaTeX math rendering and sidenotes.
